September 2020 - June 2021
Ocean Eddies

As part of the BridgeUP: STEM program at the American Museum of Natural History, our intern group created a weekly dataset of vertical ocean velocities of the North Atlantic Ocean for the past 20 years. To create this dataset, we wrote code in python to create input files for code written in fortran, and ran the code using a shell script. We visualized the dataset by creating hovmoller diagrams, allowing us to see eddies over time in a 2d image, as well as learned about machine learning for predicting eddies.
